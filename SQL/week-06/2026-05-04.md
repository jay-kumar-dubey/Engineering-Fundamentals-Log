## Q39: Movie Rating — LeetCode 1341 (Medium)
**Problem:** Find top reviewer by count (lex tiebreak) + highest avg rated movie in Feb 2020 (lex tiebreak).

**Approach:**
- Two independent ranked subqueries joined with UNION ALL
- Part 1: JOIN Users + MovieRating → GROUP BY user_id → ORDER BY COUNT(*) DESC, name ASC → LIMIT 1
- Part 2: JOIN Movies + MovieRating → filter DATE_FORMAT = '2020-02' → ORDER BY AVG(rating) DESC, title ASC → LIMIT 1
- Applied UNION vs UNION ALL distinction from prior questions — UNION ALL needed here since results are independent

**Key Pattern:** Two separate rankings into one result → UNION ALL, not UNION

**Code:**
```sql
(SELECT name AS results
FROM Users u
JOIN MovieRating mr ON mr.user_id = u.user_id
GROUP BY mr.user_id
ORDER BY COUNT(*) DESC, name ASC
LIMIT 1)

UNION ALL

(SELECT title AS results
FROM Movies m
JOIN MovieRating mr ON m.movie_id = mr.movie_id
WHERE DATE_FORMAT(mr.created_at, '%Y-%m') = '2020-02'
GROUP BY mr.movie_id, m.title
ORDER BY AVG(rating) DESC, m.title ASC
LIMIT 1);
```
